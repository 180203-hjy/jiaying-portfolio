PROJECT INSIGHT

项目背景｜从白天稻田到夜游转型

为什么要做这个项目？

01有资源

稻田景观、良渚文化、交通节点具备基础。

02有机会

余杭区过夜游客最高超 50 万人次，夜间消费近 120 亿元。

03有痛点

夜间内容不足，存在“昼热夜冷”和传播记忆点不足的问题。

CORE CONCEPT

核心策划｜月相三部曲

月相三部曲是什么？

01核心定位

稻田景观 + 良渚文化，打造差异化乡村夜游 IP。

02体验结构

以“月升·漫游、月满·体验、月落·休憩”组织夜游动线。

03策划价值

从观赏引入、文化互动到水乡停留，提升体验层次与传播记忆点。

3D MOTION DEMO

3D 动态演示｜把月相夜游方案变成可观看场景
